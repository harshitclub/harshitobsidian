## 4. Array – First Core Data Structure

Points:
- Continuous memory
- Index-based access
- Fixed structure

### Real-life use cases

1. **Photo gallery in mobile**
    - Each image has a position
    - Fast access required
    
2. **Daily temperature or stock prices**
    - Time-based data
    
3. **Game boards and grids**
    - Positions matter
    
4. **Exam scores or leaderboard**
    - Direct access needed
    
5. **Sensor data in IoT**
    - Fixed interval readings

> “Array is the fastest data structure a computer can work with.”

### Key limitation
- Insertion/deletion is expensive
- Size planning is important
